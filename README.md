## 🔧 日常维护

### 修改网页
编辑 `源文件/小饭桌宣传页.html`，然后同步到网页版：

```powershell
Copy-Item 源文件/小饭桌宣传页.html 网页版/index.html -Force
cd 网页版
git add . ; git commit -m "更新内容" ; git push
```

### 修改小程序
在微信开发者工具中打开 `微信小程序/` 文件夹，修改后直接上传。

### 添加新菜品图片
1. 放入 `原始素材/`
2. 压缩后放入 `网页版/小饭桌广告素材/`
3. 在微信小程序中更新 CDN 图片列表（`pages/index/index.js` 中的 `foodImages`）
